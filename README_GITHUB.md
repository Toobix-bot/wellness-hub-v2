## 🚀 **Schnellstart**

### **Lokale Entwicklung:**
```bash
# Repository klonen
git clone https://github.com/yourusername/wellness-hub-framework.git

# Dependencies installieren  
npm install

# Development Server starten
npm run dev

# Build für Production
npm run build
```

### **Deployment:**
```bash
# Auf GitHub pushen
git add .
git commit -m "Update"
git push origin main

# Vercel deployed automatisch! 🎉
```
